## üèóÔ∏è Basic Concepts

### Core Components

- **Prompts** - Reusable prompt templates with variables
- **Variables** - Dynamic content substitution using `{{variable}}` syntax
- **Models** - AI model-specific prompt optimization
- **Caching** - Multi-level caching for performance
- **Telemetry** - Built-in usage tracking and analytics

### Authentication Flow

PromptOps uses API key authentication with HMAC signatures:

1. **API Key** - Your unique identifier
2. **Secret Key** - Used for HMAC signature generation
3. **Timestamp** - Prevents replay attacks
4. **Signature** - Ensures request integrity
